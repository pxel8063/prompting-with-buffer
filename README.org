* Overview

=pwb= is a lightweight Emacs client for interacting with Anthropic's Claude API.
It allows you to send buffer contents to Claude and receive AI-generated responses directly within Emacs.

** Features

- Send buffer content as prompts to Claude API
- Maintain conversation history across sessions
- Customize system prompts
- Display responses in a dedicated =*Anthropic*= buffer
- Support for Claude model selection and token limits

* Prerequisites

- GNU Emacs 28.1 or later
- =curl= command-line tool
- Anthropic API key (obtain from [[https://console.anthropic.com][Anthropic Console]])

* Installation

** Step 1: Clone the repository

#+BEGIN_SRC bash
git clone https://github.com/pxel8063/prompting-with-buffer.git ~/.emacs.d/lisp/
#+END_SRC

** Step 2: Add to init.el

#+BEGIN_SRC elisp
(add-to-list 'load-path "~/.emacs.d/lisp/prompting-with-buffer")
(require 'pwb)
#+END_SRC

** Step 3: Set API Key
Set =auth-sourc= for the API.
#+BEGIN_SRC bash
machine api.anthropic.com password s*-*t-a-*api-key**
#+END_SRC

* Usage

** Basic Workflow

1. Write or select text in an Emacs buffer
2. Run =M-x pwb-current-buffer=
3. (Optional) Enter prefill text when prompted
4. View Claude's response in the =*Anthropic*= buffer

** Commands

| Command                   | Description                                          |
|---------------------------+------------------------------------------------------|
| =pwb-current-buffer=      | Send the content of the current buffer to Claude API |
| =pwb-set-system-prompt=   | Define system prompt from current buffer             |
| =pwb-clear-system-prompt= | Clear the current system prompt                      |
| =pwb-clear-messages=      | Reset conversation history                           |

** Customizable Variables

| Variable                       | Default                                 | Description                       |
|--------------------------------+-----------------------------------------+-----------------------------------|
| =pwb-claude-model=             | ="claude-haiku-4-5"=                    | Claude model to use               |
| =pwb-claude-max-tokens=        | =1024=                                  | Maximum tokens in response        |
| =pwb-claude-api-host=          | ="api.anthropic.com"=                   | The machine name in =authxsource= |
| =pwb-claude-anthropic-version= | ="2023-06-01"=                          | Anthropic version                 |
| =pwb-claude-message-api-url=   | "https://api.anthropic.com/v1/messages" | The message API url               |
| =pwb-claude-system-prompt=     | "When possible, use org-mode syntax."   | The system prompt                 |

*** Modify via Customize

#+BEGIN_SRC elisp
M-x customize-group RET pwb RET
#+END_SRC

Or in init.el:
#+BEGIN_SRC elisp
(setq pwb-claude-model "claude-3-5-sonnet-20241022")
(setq pwb-claude-max-tokens 2048)
#+END_SRC

* Examples

** Example 1: Basic Prompt

1. Create a new buffer or open existing file
2. Type your prompt:
   #+BEGIN_EXAMPLE
   Write a Python function that calculates the nth Fibonacci number
   #+END_EXAMPLE

3. Run =M-x pwb-current-buffer RET RET=
4. View the response in =*Anthropic*= buffer

** Example 2: Set System Prompt

1. Create a new buffer
2. Enter your system prompt:
   #+BEGIN_EXAMPLE
   You are an expert Python developer. Provide concise, 
   efficient code with clear explanations.
   #+END_EXAMPLE

3. Run =M-x pwb-set-system-prompt=
4. All subsequent prompts will use this system context

** Example 3: Multi-turn Conversation

1. Send first prompt using =pwb-current-buffer==
2. View response in =*Claude*= buffer
3. Open another buffer with follow-up prompt
4. Run =pwb-current-buffer= again
5. Conversation history is automatically maintained
